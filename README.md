## Ship to Wikitext
# Extract ship data
* paste the contents of the built-in folder into the /png folder
* run cosmoteer_save_tools.py
* this will output 2 files, ship-parts.json with only parts & ship-data.json with all the data

# Create wikitext files
* open cosmoteer_create_page.py
* fill in non-ship file information
* run cosmoteer_create_page.py
* this should generate [shipname]wikitext.txt files inside the /wikitext folder
* copy paste these into new wiki pages for those ships on [cosmoteer.wiki.gg](https://cosmoteer.wiki.gg/)
